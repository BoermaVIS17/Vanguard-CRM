# Image Upload Instructions

## Roof Measurement Guide Image

Please save the roof measurement reference image as:
**`roof-measurement-guide.png`**

This image should show the color-coded roof components diagram with:
- Red (Gutters/Eaves)
- Blue (Gables/Rakes)  
- Green (Valleys)
- Yellow (Ridges)
- Orange (Hips)
- Purple (Flashing)

The image will be displayed in the Manual Roof Takeoff section as a reference guide for users.
